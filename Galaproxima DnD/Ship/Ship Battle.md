Ship battles are on a hex grid.

Each ship in the battle has a movement action and generic action for each turn.

**Movement:**
	Ships can only move forward, but in a movement action whoever's piloting can turn the ship which changes the ships direction.
	Each ship has a 'throttle' which determines how many tiles forward the ships moves on it's turn.
		Full throttle:
			The ship's max speed
		Med throttle:
			The ship's medium speed
		Low throttle:
			The ship's minimum speed

**Combat:**
	During a ship's turn and if the ship has weapons the pilot can use them against the enemy ship.
	The enemy ship has to be 'in view' of the side of the ship the weapon is on in order to attack. 
	When a weapon is used the user must roll to hit and if the roll is higher than the enemy ship's hull's AC it hits.

**Escape:**
	If things are getting a little dicey the pilot can try to escape by spooling the FTL drive and making a jump, spooling the drive takes 4 turns before you can make the jump.
